The Sapphire RMS Plugin for WordPress is designed for existing Sapphire customers that want to integrate reservation functionality into their website.

== Description ==

Plese visit http://wpplugin.sapphirerms.com for the latest information.

== Installation ==

Installation is no different than any other WordPress plugin.

1. Make sure to have Twitter Bootstrap Plugin Installed.
2. Upload the contents of `SapphireRMS.zip` into the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Configuration can be performed via the Settings, Sapphire RMS menu option.
